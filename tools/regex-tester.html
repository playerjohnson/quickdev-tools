How to Use

Type a regular expression in the pattern field at the top. Toggle flags with the g i m s buttons. Enter or paste your test string in the left panel. Matches highlight instantly in the right panel, and full match details including capture groups appear in the table below.

Regex Quick Reference

. any character, \d digit, \w word char, \s whitespace, ^ start of line, $ end of line, * 0+, + 1+, ? 0 or 1, {n,m} between n and m, [abc] character class, (group) capture group, (?:group) non-capturing, a|b alternation.

JavaScript Regex Flavour

This tester uses the JavaScript RegExp engine. It supports lookbehind (?<=...), named groups (?<name>...), and the s (dotall) flag. Some features from PCRE (like recursive patterns) are not available in JavaScript.
